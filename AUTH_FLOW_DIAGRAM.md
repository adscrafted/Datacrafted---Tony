## Quick Reference: When to Use What

### Use AuthModal
- On landing page
- User-initiated sign in (button click)
- Can be closed/dismissed
- Used when auth is optional

### Use AuthGateModal
- On protected routes
- Automatic display (not user-initiated)
- Cannot be dismissed without authenticating
- Used when auth is required

### Use LandingHeader
- On landing page only
- Shows different states based on auth
- Triggers AuthModal when needed

### Use MinimalHeader
- On authenticated pages (projects, dashboard)
- Simple logo + navigation
- No auth controls needed
